# タイピングれんしゅう（小1〜小6）

小学生向けのタイピング練習（静的サイト）。

- ① ひらがな（ローマ字でうつ）
- ② アルファベット（A〜Z）
- ③ ローマ字（たんご/ぶんしょう）

## 公開（GitHub Pages）

このリポジトリを GitHub Pages で公開すれば、そのURLに飛ぶだけで使えます。

## 操作

- コース/レベル/問題数を選んでスタート
- ひらがな/アルファベット：Enterなし
- ローマ字：Enterなし（正しいキーだけ進む／スペースは自動スキップ）

## 保存

- 記録は localStorage に保存されます（端末ごと）
